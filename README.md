# Zelun Huang

https://zelunhuang.github.io/

## 内容

- 首页：`index.html`
- 关于页：`about.html`
- 样式：`css/style.css`
- 脚本：`js/main.js`
- 图片资源：`images/avatar.jpg`
- 其他资源：`assets/`

## 更新方式

代码改动后，进入 `D:\MyWebsite` 执行：

```powershell
git add .
git commit -m "Update website files"
git push
```

如果远程仓库又有新提交：

```powershell
git pull origin main --rebase
git push
```
