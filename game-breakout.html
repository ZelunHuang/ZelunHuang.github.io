Breakout

Break the bricks — with gravity.

Score: 0 Lives: 3 Best: 0

Rules
Break all the bricks using the ball. The ball is affected by gravity — it accelerates downward, creating parabolic arcs. Each brick colour is worth a different score (1–7 pts). You have 3 lives. Losing all lives or clearing all bricks ends the game.
