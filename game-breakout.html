Basketball

Shoot hoops — with gravity.

Score: 0 Streak: 0 Best: 0
Drag from ball to aim (opposite direction)  ·  Release to shoot  ·  P pause
Rules
Drag away from the hoop to set power & angle — like a slingshot. Pull further = shoot harder. The ball follows a gravity arc. Each basket: +10 pts, consecutive baskets build a streak multiplier (×2, ×3…). The hoop moves to a new random position after each attempt. Miss and you start over.
